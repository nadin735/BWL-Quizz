BWL Quiz

Simple quiz game written in Python.

The program asks different questions about business and economics basics and calculates the final score at the end.

Features:
Multiple quiz questions
Score tracking
Percentage result
Simple terminal interaction

Requirements:
Python 3

Run the Project:
python bwl_quiz.py
Project Goal

Practice basic Python concepts like:
variables
user input
conditions
if statements
simple program structure

Future Ideas:
